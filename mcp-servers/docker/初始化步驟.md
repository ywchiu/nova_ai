# 本機測試環境 — 初始化步驟

## 前置需求

- Docker Desktop 已安裝並啟動
- 至少 6GB 可用記憶體
- 至少 20GB 可用硬碟空間

---

## 一、啟動所有服務

```bash
cd docker/
docker compose up -d
```

等待約 2-3 分鐘讓服務完全啟動。查看啟動狀態：

```bash
docker compose logs -f        # 看全部 log
docker compose logs -f jira   # 只看 Jira log
```

---

## 二、初始化 Jenkins（最快，約 5 分鐘）

### 1. 取得初始密碼

```bash
docker exec nova-jenkins cat /var/jenkins_home/secrets/initialAdminPassword
```

複製輸出的密碼。

### 2. 開啟瀏覽器

前往 `http://localhost:9090`，貼上密碼。

### 3. 安裝建議套件

選 **Install suggested plugins**，等待安裝完成。

### 4. 建立管理員帳號

- 帳號：`admin`
- 密碼：`admin123`（測試用，隨便設）

### 5. Jenkins URL 設定

保持預設 `http://localhost:9090/`，按 Save and Finish。

### 6. 取得 API Token

1. 右上角帳號 → **Configure**
2. **API Token** → **Add new Token** → 輸入名稱 `mcp-test` → **Generate**
3. **複製 token（只顯示一次！）**

---

## 三、初始化 Jira（約 10-15 分鐘）

### 1. 開啟瀏覽器

前往 `http://localhost:8080`

### 2. 選擇安裝類型

選 **「我自己設定」（I'll set it up myself）**

### 3. 資料庫設定

已在 docker-compose 設定好，直接按 **下一步**。

### 4. 取得試用 License

點 **「取得 Jira 評估 License」**，Atlassian 網站登入後會自動填入（30 天免費）。

> 需要 Atlassian 帳號（免費申請：https://id.atlassian.com）

### 5. 建立管理員帳號

- 帳號：`admin`
- 密碼：`admin123`
- Email：任意

### 6. 建立測試專案

1. 登入後 → **建立專案**
2. 選 **Scrum**
3. 專案名稱：`Nova Test`，Key：`NOVA`

### 7. 取得 PAT

1. 右上角頭像 → **Profile**
2. 左側 → **Personal Access Tokens** → **Create token**
3. 名稱：`mcp-test`，**複製 token**

---

## 四、初始化 BitBucket（約 10 分鐘）

### 1. 開啟瀏覽器

前往 `http://localhost:7990`

### 2. License 設定

同 Jira，點 **「取得評估 License」**（需要 Atlassian 帳號）。

### 3. 建立管理員帳號

- 帳號：`admin`
- 密碼：`admin123`

### 4. 建立測試 Project 和 Repo

1. 左側 → **Projects** → **Create project**
2. 名稱：`Nova`，Key：`NOVA`
3. 進入 project → **Create repository**
4. 名稱：`nova-backend`

### 5. 取得 HTTP Access Token

1. 右上角頭像 → **Manage Account**
2. 左側 → **HTTP Access Tokens** → **Create token**
3. 名稱：`mcp-test`，權限選 `Project admin`
4. **複製 token**

---

## 五、設定 MCP Server 的 .env

在 `mcp-servers/` 目錄建立 `.env` 檔（只用於測試，不 commit）：

```bash
# mcp-servers/.env

JIRA_BASE_URL=http://localhost:8080
JIRA_PAT=貼上你的Jira PAT

BB_BASE_URL=http://localhost:7990
BB_PAT=貼上你的BitBucket Token

JENKINS_URL=http://localhost:9090
JENKINS_USER=admin
JENKINS_API_TOKEN=貼上你的Jenkins API Token
```

---

## 六、測試 MCP Server

### 用 MCP Inspector 測試（需先安裝 Node.js）

```bash
# 安裝 inspector（只需一次）
npm install -g @modelcontextprotocol/inspector

# 測試 Jira
source ../.env  # 載入環境變數
npx @modelcontextprotocol/inspector python3 ../jira_mcp/server.py

# 測試 Jenkins（最容易，不需要 License）
npx @modelcontextprotocol/inspector python3 ../jenkins_mcp/server.py
```

開啟 `http://localhost:5173`，點選工具，填入參數，按 **Execute**。

### 用指令直接測試（最簡單）

```bash
# 確認 Jira 連線
curl -H "Authorization: Bearer 你的PAT" \
     http://localhost:8080/rest/api/2/project

# 確認 Jenkins 連線
curl -u admin:你的APIToken \
     http://localhost:9090/api/json
```

## 常用指令

```bash
# 啟動
docker compose up -d

# 停止（保留資料）
docker compose down

# 重啟單一服務
docker compose restart jira

# 查看記憶體使用量
docker stats

# 完全清除（資料也刪）
docker compose down -v
rm -rf jira-data/ bitbucket-data/ jenkins-data/ postgres-data/
```
