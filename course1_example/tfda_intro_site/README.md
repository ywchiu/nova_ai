# 台灣食品藥物管理署 (TFDA) 介紹網站

## 專案說明

本專案為**教學示範用途**之靜態網站，用於展示如何使用純 HTML、CSS 和 Vanilla JavaScript 建構政府資訊服務類型的網站。

> ⚠️ **重要聲明：本網站為教學示範用途，非台灣食品藥物管理署 (TFDA) 官方網站。所有內容僅供學習參考，不構成任何法律依據。**

## 目錄結構

```
tfda_intro_site/
├── index.html      # 主要 HTML 頁面
├── styles.css      # CSS 樣式表（含響應式設計）
├── script.js       # JavaScript 互動腳本
└── README.md       # 本說明文件
```

## 功能特色

### 頁面內容
- **機關任務**：展示 TFDA 的核心使命與四大核心價值
- **四大業務領域**：食品、藥品、醫療器材、化粧品
- **民眾服務入口**：6 大服務管道快速連結
- **安全提醒**：食品安全與藥品安全即時公告
- **最新公告**：分類展示最新政策與法規動態
- **常見問題**：FAQ 展開收合互動
- **聯絡資訊**：辦公資訊與線上申辦服務

### 互動功能
1. **響應式導航選單**：手機版漢堡選單
2. **全站搜尋**：即時搜尋與關鍵字高亮
3. **業務主題切換卡**：點擊切換四大業務領域
4. **公告篩選**：依類別篩選最新公告
5. **FAQ 展開收合**：展開/收合常見問題
6. **平滑滾動**：錨點連結平滑滾動效果

### 設計特點
- 正式、乾淨、可信任的政府網站風格
- 完整響應式設計（桌面、平板、手機）
- 無外部 CDN 或網路資源依賴
- 支援無障礙功能（ARIA 屬性）
- 支援 `prefers-reduced-motion` 無障礙偏好

## 技術規格

| 項目 | 規格 |
|------|------|
| HTML | HTML5 |
| CSS | CSS3（CSS 變數、Grid、Flexbox、Media Queries） |
| JavaScript | ECMAScript 6+（無框架依賴） |
| 字體 | Microsoft JhengHei, PingFang TC, Noto Sans TC |
| 響應式斷點 | 1024px（平板）、768px（手機）、480px（小螢幕） |

## 啟動方式

### 使用 Python 內建伺服器

```bash
cd tfda_intro_site
python3 -m http.server 8080
```

啟動後可在瀏覽器開啟：`http://localhost:8080`

### 直接使用瀏覽器

直接在檔案管理器中雙擊 `index.html` 即可開啟。

## 瀏覽器相容性

- Chrome / Edge 90+
- Firefox 88+
- Safari 14+

## 教學應用

本專案適合用於以下教學場景：

1. **HTML 結構設計**：政府網站的語意化標籤使用
2. **CSS 布局技術**：Grid、Flexbox、CSS 變數、響應式設計
3. **JavaScript 互動**：事件處理、DOM 操作、無框架開發
4. **無障礙設計**：ARIA 屬性、鍵盤操作支援
5. **響應式設計**：多斷點適配策略
