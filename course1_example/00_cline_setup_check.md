# Cline：課前環境 Smoke Test

> 對應投影片：開始實作前

把這段貼到 Cline，先確認模型、provider、工作目錄都可用。這一步不應該讀寫檔案，也不應該執行 terminal command。

---

請只做一個 Cline 環境 smoke test。

限制：

- 不要讀取任何檔案
- 不要建立或修改任何檔案
- 不要執行 terminal command
- 不要開始實作課程專案

請回覆：

1. 目前 Cline 使用的 provider 名稱
2. 目前 Cline 使用的 model 名稱
3. 目前 workspace folder 名稱
4. 最後一行輸出：`Cline ready`

如果 provider 或 model 連線失敗，請停在錯誤訊息，不要繼續跑後面的課程範例。
