# Taiwan Food and Drug Administration (TFDA) Introduction Site - English Version

An educational demonstration website introducing the Taiwan Food and Drug Administration (TFDA). Built with pure HTML, CSS, and Vanilla JavaScript — no external dependencies or CDN resources required.

## Features

- **Agency Mission**: Overview of TFDA's core responsibilities and values
- **Four Business Areas**: Interactive tabs covering Food Safety, Drug Management, Medical Devices, and Cosmetics
- **Public Service Portals**: Quick access to food licenses, drug information, complaint hotlines, and more
- **Safety Alerts**: Food and drug safety announcements and reminders
- **Latest Announcements**: Filterable news by category (Food, Drugs, Devices, Cosmetics)
- **FAQ Accordion**: Expandable frequently asked questions section
- **Contact Information**: Office address, phone, email, service hours, and online application services
- **Site Search**: Full-text search with keyword highlighting
- **Responsive Design**: Fully responsive layout for desktop, tablet, and mobile devices

## Usage

### Start Local Server

Using Python's built-in HTTP server:

```bash
cd tfda_intro_site_en
python3 -m http.server 8082
```

Then open http://localhost:8082 in your browser.

### Manual Inspection

Simply open `index.html` directly in any modern web browser.
